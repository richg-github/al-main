Upgrades and Migrations

Alert Logic aims to make the upgrade process as simple and seamless as possible, offering full feature parity for upgraded customers. Resources are available to guide you through your upgrade and help you take advantage of the upgraded and new features on the Managed Detection and Response (MDR) platform.

Console upgrade

  • Upgrade to Managed Detection and Response
  • Migration Prechecks
  • Recommended Training for New and Upgrading Customers

Functionality upgrades

Available with all MDR subscriptions:

  • Machine Learning Log Review Upgrade (March 2021)
  • Incidents Upgrade (March 2021)
  • Log Search Upgrade (February 2021)
  • Cloud Defender Omnibox Search Upgrade (February 2021?)
  • Notifications Upgrade (February 2021)
  • Scan Functionality Upgrade (January 2021)
  • Managed Detection and Response Reports Upgrade (February 2019)

Additional features

Available with all MDR subscriptions: 

  • CIS Microsoft Azure Foundation Benchmark reporting (~March 2020)
  • Dashboards (February 2020)
  • Threat Risk Index (February 2020)
  • Exposures (February 2020)
  • Extended Endpoint Protection (September 2019
  • Health (April 2019)
  • Cross-Network Protection (Data Center deployments) (April 2019)

Available with Alert Logic MDR Professional and Alert Logic MDR Enterprise subscriptions:

  • File Integrity Monitoring (September 2020)
  • Web Log Analytics (WLA) (September 2020)
  • Application Registry (April 2020)
  • Policy-based configuration of system and application (flat file) logs
  • Improved support for Azure deployments and AWS deployments
